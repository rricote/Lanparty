3.1 Font Awesome

It’s one of the most powerfull tools. Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS. Let’s take a look. In the forum.html you can find few icons of stars on the left. This is a font awesome so let’s say we wanna change them to more suitable for your purposes. Open the forum.html in some editor and find the line number 288.

Here you can change appearance of this icon just simply remove class „fa-star“ and type for instance „fa-money“. Now save the changes and refresh the index_normal.html page and icon is different.

For full documentation how to customize this plugin (size, color, rotation etc.) and full list of different types of icons go to fontawesome.io

3.5 Twitter Post Fetcher

This plugin is used on the bottom of sidebar in index.html section and displays your latest tweets. You only have to paste your Twitter Widget ID to connect your Twitter account. To do it go to custom.js file and paste your twitter widget ID on the line 107

How to find your Twitter Widget ID is described here

For mor information how display your tweets check full documentation of this plugin. Twitter Post Fetcher

3.6 Vegas

Vegas is plugin to set up a background with different types of overlays in our theme. You can find complete documentation here

We prepared for you 2 options, first is static image and second is slideshow. There is lot of other options which you can find in plugin's documentation. Default option is slideshow

We used slideshow of 5 images. If you want to change these images just go to "images" folder and replace them. You can also change overlay pattern (line 40) in custom.js.

If you want to use only static image just delete the previous code of initializing slideshow and remove this two characters // befor line 29 in custom.js

3.7 jQuery Countdown

As the name of this plugin says this is just simple countdown. This plugin you can find on tournament.html

If you want to change date just simply open custom.js and go to line 19 and change '2016/10/10' to any date you want in format 'YYYY/MM/DD'

For mor information how display your tweets check full documentation of this plugin. jQuery Countdown

3.8 Animate

It is awesome CSS3 plugin which allows us to animated individal parts of our themes. You can find animate effect e.g on index.html when you click to "search" button on the top right corner.

For full documentation go to Animate css

4. FAQ

If I buy an Single application license, can I remove your Copyright-Line?

Yes, in any kind of license you definitely can remove copyright. But if you like our work you can place there just "created by pixelized" it is completely up to you.

How do i connect contact form with my email.

This is only html5, css3 & jQuery theme. There isn't any script to do it. You have to use some external script probably PHP or something. We are selling many of copies and everybody has slightly different requirement so we are not able to build single one script.

Nevertheless we have prepared for you little example how can look like PHP script for sending messages. For more informations check PHP mailer.

Can i use, images included in this theme?

In credits section you can find links to every image. So find image you want in credits and check licence on source page.
